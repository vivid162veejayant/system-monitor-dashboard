# 🖥️ System Monitor Dashboard

A browser-based system monitoring dashboard built using **HTML, CSS, and JavaScript**.

This project simulates system metrics such as CPU, RAM, Disk, and GPU usage with realistic behavior including smooth transitions, spikes, and system load patterns.

---

## 🚀 Features

- 📊 Live CPU, RAM, Disk, and GPU simulation
- 📈 Real-time graphs using Chart.js
- 🎯 Smooth and realistic usage behavior (no random spikes)
- ⚙️ System modes (Idle, Normal, Heavy)
- 🧠 State-based simulation engine
---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Chart.js

## ▶️ How to Run

1. Download or clone this repository
2. Open `index.html` in your browser

No installation or dependencies required.

## ⚠️ Disclaimer

This project is a **simulation only**.

- It does NOT fetch real system data
- All metrics are generated using custom JavaScript logic
